# 你画我猜 AI游戏

一个基于AI的在线你画我猜游戏，玩家可以在画布上作画，AI系统会实时识别并猜测画作内容。

## 技术栈

- 前端：React + TypeScript
- 构建工具：Vite
- 画布：HTML5 Canvas
- 后端：Node.js + Express
- AI：OpenAI Vision API

## 功能特点

- 实时绘画功能
- AI智能识别
- 即时反馈
- 简洁的用户界面
